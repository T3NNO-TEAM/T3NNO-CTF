# 🚀 T3NNO-CTF
### 🛠️ CTF Tool v0.1

Console-based toolkit for CTF players and beginners in cybersecurity.

---

## ✨ Features

- 🔓 Brute Force Hash
- 📦 Brute Force ZIP
- 🌐 Port Scanner
- 🕸️ Web Scanner
- 📁 File Hash Scanner (Get Files HASH)

---

## ⚙️ Requirements

- Python 3.9+
- pip

---

## 📦 Installation

```bash
# Clone the repository
git clone https://github.com/your-username/T3NNO-CTF.git
cd T3NNO-CTF

# Create virtual environment (recommended)
python -m venv venv

# Activate virtual environment
# Windows:
venv\Scripts\activate
# Linux / Mac:
source venv/bin/activate

# Install dependencies
pip install -r requirements.txt


## ▶️ Usage

Run the console app:

```bash
python main.py

Follow the interactive menu to select the module you want:

----------- menu -----------

[1] Brute Force Hash
[2] Brute Force ZipFile
[3] Port Scanner
[4] Web Scanner
[5] Get Files HASH

Enter the number of the option and follow the prompts for each module.

Tip: Press Ctrl + C to exit gracefully.

🧩 Modules
brute_force_hash → Brute Force Hash
brute_force_zip → Brute Force ZIP
port_scaner → Port Scanner
